Phase 2 — navigation prototypes

Two ways to structure the 5 tabs. Click each, navigate between tabs, watch the URL bar, decide which feels right.

A · Single-page with URL hashes

One index.html. Tab switching is instant (JS show/hide). URL updates via #hash.

/us-ai-infra/#investor

B · Multi-page (separate HTML files)

One file per tab. Tab switching triggers a full page reload. Browser back/forward works natively.
